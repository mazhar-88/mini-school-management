# Mini School Management System

A full-stack mini school application focused on managing students, teachers, classes, and sections.

---

## Clone Project
git clone https://github.com/your-username/mini-school-management.git

---

## Setup Frontend
cd frontend  
npm install  

---

## Setup Backend
cd backend  
npm install  

---

## Database Setup

This project uses **MySQL** as the database.

- Database is configured using environment variables
- You can use **XAMPP / phpMyAdmin** or any local MySQL server
- Create a database before running the backend

### Required Environment Variables (Backend)

Create a `backend/.env` file and add:

DB_HOST=localhost
DB_USER=root
DB_PASS=your_password
DB_NAME=mini_school_db
PORT=5000


---

### Start Backend
cd backend  
npm run dev  

### Start Frontend
cd frontend  
npm run dev
