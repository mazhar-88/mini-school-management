# Mini School Management System

A mini school application focused on managing students, teachers, classes, and sections.

## Clone Project
git clone https://github.com/your-username/mini-school-management.git

## Setup Frontend
cd frontend  
npm install  

## Setup Backend
cd backend  
npm install  

## Start Project

### Start Backend
cd backend  
npm run dev  

### Start Frontend
cd frontend  
npm start
